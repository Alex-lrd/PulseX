# 📡 Projet Radar  
UE : Électronique & Communication Sans Fil

## 📖 Description  
Ce projet consiste en la réalisation d’un système de type radar permettant de détecter des obstacles à distance à l’aide d’un capteur ultrasonique.  
Le système collecte des données, les traite, puis peut les exploiter pour des applications comme la détection d’objets ou la cartographie simple de l’environnement.

## 🎯 Objectifs  
- Mesurer des distances avec précision  
- Traiter et exploiter les données de détection  
- Mettre en œuvre une communication sans fil (si applicable)  
- Comprendre l’intégration matériel + logiciel  

## 🧩 Composants utilisés  
- Carte UCA  
- Cerveau moteur  
- Capteur ultrasonique (ex : HC-SR04)  

## ⚙️ Technologies utilisées  
- Programmation embarquée (C / C++ / Arduino)  
- Traitement du signal  
- Communication sans fil (LoRa)  
- Électronique embarquée  

## 🛠️ Fonctionnement  
1. Le capteur ultrasonique émet une onde sonore  
2. L’onde est réfléchie par un obstacle  
3. Le capteur mesure le temps de retour  
4. La distance est calculée avec la formule :  

   distance = (temps × vitesse du son) / 2

5. Le système peut effectuer un balayage grâce au moteur  

## 📊 Résultats attendus  
- Détection fiable des obstacles  
- Visualisation des distances (console, graphique, etc.)  
- Balayage de zone  

## 🚀 Améliorations possibles  
- Ajout d’une interface graphique  
- Transmission des données en temps réel  
- Amélioration de la précision  
- Ajout d’autres capteurs
